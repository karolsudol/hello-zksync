# hello-zksync

zksync2 contract hardhat deployment

## deploy

`npm init -y`
`npm i -D typescript ts-node ethers zksync-web3 hardhat @matterlabs/hardhat-zksync-solc @matterlabs/hardhat-zksync-deploy`
`npx hardhat compile`
`npx hardhat deploy-zksync`
